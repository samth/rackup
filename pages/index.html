rackup curl | sh

Install the rackup toolchain manager bootstrap script directly from GitHub Pages. This installs the manager and a hidden internal runtime (but not a user Racket toolchain yet).

Install
curl -fsSL https://samth.github.io/rackup/install.sh | sh
View install.sh Repository
Notes
  • The bootstrap installs a hidden internal runtime so a preinstalled system racket is not required.
  • The bootstrap script prompts before editing shell config unless you pass -y.
  • After bootstrap: run rackup install stable --set-default.
Noninteractive
curl -fsSL https://samth.github.io/rackup/install.sh | sh -s -- -y

Tip: the site also serves /install as an alias to /install.sh.
