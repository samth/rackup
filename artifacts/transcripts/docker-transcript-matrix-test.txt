rackup - Racket toolchain manager

Commands:
  available [--all|--limit N]  List remote install specs and recent release versions.
  install <spec> [flags]  Install a Racket toolchain (release, pre-release, snapshot).
  link <name> <path> [flags]  Link an in-place/local Racket build as a managed toolchain.
  list                  List installed toolchains (shows default/active tags).
  default [id|status|set <toolchain>|clear|<toolchain>|--unset]  Show, set, or clear the global default toolchain.
  current [id|source|line]  Show the active toolchain and where it came from.
  which <exe> [--toolchain <toolchain>]  Show the real executable path for a tool in a toolchain.
  shell <toolchain> | shell --deactivate  Emit shell code to activate/deactivate a toolchain in this shell.
  run <toolchain> -- <command> [args...]  Run a command using a specific toolchain without changing defaults.
  prompt [--raw|--source]  Print fast prompt info for PS1 (wrapper handles this without Racket startup).
  remove <toolchain>    Remove an installed or linked toolchain and its addon dir.
  reshim                Rebuild executable shims from installed toolchains.
  init [--shell bash|zsh]  Install/update shell integration in ~/.bashrc or ~/.zshrc.
  uninstall [--yes]     Remove rackup, its toolchains/runtime, and shell init blocks (destructive).
  self-upgrade [--with-init]  Upgrade rackup's code by rerunning the installer into the current RACKUP_HOME.
  runtime status|install|upgrade  Manage rackup's hidden internal runtime used to run rackup itself.
  doctor                Print diagnostics for paths, runtime, and installed toolchains.
  help [command]        Show global help or help for a specific command.
